# Kira

Kira is an open source diary platform built on react native. There are two main scenes. 
## LoginScene

The first scene is the LoginScene. This is where a user can either log in or sign up.

![LoginScene](http://i.giphy.com/3oKIPjqg3fIvi1oowg.gif)


## DiaryScene

Once the user is logged in the user will be presented with three tabs. A tab for personal diary entries, a tab to write diary entries, and a tab to see other people's diary entries.

![DiaryScene](http://i.giphy.com/3og0IATwWqK6m3yJCU.gif)
